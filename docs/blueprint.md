# **App Name**: PlastiTrack

## Core Features:

- Plastic Logging: Allow users to log the type, quantity, and weight of plastic they collect.
- Location Tracking: Enable users to track locations where they find and dispose of plastic using a map interface.
- Global Impact Visualization: Display aggregated user data to show the total amount of plastic saved, CO2 reduced, and water conserved.
- Weekly Overview and Tips: Provide weekly statistics and tips to help users reduce their plastic consumption, perhaps with recipe recommendations generated by a tool using generative AI.
- Reward System: Implement a badge and points system to reward users for their plastic collection efforts.

## Style Guidelines:

- Primary color: Forest green (#386641) to evoke nature and environmental friendliness.
- Background color: Light sage (#E3E8E4), very desaturated hue from the same green as the primary, for a natural calming base.
- Accent color: Mustard yellow (#D4AC0D) for highlights, CTAs, and badges, providing a sunny and optimistic contrast.
- Body and headline font: 'PT Sans' for a balance of modern look and friendly feel, suitable for both.
- Use nature-inspired icons with a line art style.
- Maintain a clean and organized layout with rounded corners and ample spacing to enhance readability and user experience.
